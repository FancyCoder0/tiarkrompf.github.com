Update: I'm joining Purdue University as assistant professor in fall 2014.

I'm a post-doc at Oracle Labs and EPFL. My research focus is on generative programming, which can turn high-level programs into highly specialized and efficient implementations.

I'm interested in all aspects of programming, in particular programming languages, parallelism and compilers, with a focus on making very high-level programming abstractions more efficient. In general I am interested in bringing theoretical insights into practical use.

Most of my recent work revolves around Lightweight Modular Staging (LMS) and Delite, technologies for integrating runtime code generation into high-level programs and building embedded compilers for domain specific languages (DSLs). I initiated and still lead the EPFL side of this work, which is a joint effort between Martin Odersky's group at EPFL and Kunle Olukotun's group at the Stanford Pervasive Parallelism Lab. At Oracle Labs I have started Project Lancet, which aims to turn JIT compilers into precision tools in future JVM generations. I'm also looking into applying LMS to speed up data processing and query engines.

As a member of Martin Odersky's Scala team since 2008, my contributions to Scala include delimited continuations, efficient immutable data structures (Set, Map, Vector) and speedups in the Scala compiler (type checking, implicit search, bytecode generation; up to 3x for certain projects). I also started the Scala-Virtualized research branch. More recently I became involved in formalizing parts of Scala's type system in the DOT calculus.

I received an MS in computer science from University of Lübeck (2008) and a PhD from EPFL (2012). Before my PhD I co-founded two startups and lead software projects outside of academia, ranging from CAD software for kitesurfing equipment to hospital systems for electronic anesthesia records, image processing for OCT workstations, and therapeutic games.

Projects

  • Lightweight Modular Staging (LMS): runtime code generation and embedded compilers (abstraction without regret).
  • Project Lancet: surgical precision JIT compilers.
  • Delite: a compiler and runtime framework for compiled embedded DSLs.
  • Scala-Virtualized: redefining built-in language features as library calls (language virtualization and deep linguistic reuse).

Past projects:

  • Vodka: a concurrent programming language based on join patterns and multi-method dispatch.
